Überblick Mitmachen Hardware Software Anwendung

Das IPS-System wird fortwährend verbessert und weiterentwickelt.

Die neueste stabile Version der IPS-Software kann über den Download-Button links heruntergeladen werden. Alternativ ist diese Version auch im Github-Repository unter dem Branch "Master" zu finden.

Um in die Entwicklung einzusteigen, kannst du die IPS-Software im aktuellen Entwicklungszustand unter dem Branch "Develop" herunterladen.

Möchtest du Veränderungen an dieser Präsentationsseite vornehmen, so ist dies durch Editieren der Html-files im Branch "gh-pages" möglich.

Sind deine Ideen und Verbesserungen an Konzept oder Code nachvollziehbar und funktionabel, so werden sie in den Master-Branch übernommen. Vorraussetzung hierfür ist jedoch auch eine gute Dokumentation der Entwicklung und verständlich dokumentierter Code.

Github lässt sich auch leicht über die Konsole bedienen. Hier ist eine Liste der wichtigsten Befehle: 

git clone https://github.com/Daedalus-TUM/ips-indoor-navigation [branch]
läd den Branch [branch] herunter, falls dieser im lokalen Ordner noch nicht existiert.

git pull https://github.com/Daedalus-TUM/ips-indoor-navigation [branch]
läd den Branch [branch] herunter und überschreibt lokale Dateien.

git add [file]
git commit -m [message]
git push https://github.com/Daedalus-TUM/ips-indoor-navigation [branch]
Die Befehle add, commit und push sind notwendig um eine lokale Datei auf die Github-Repository hochzuladen. Durch eine kurze Nachricht [message] kannst du mitteilen, was der Zweck dieser Veränderung ist.
